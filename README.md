# Python — Programação Orientada a Objetos

Repositório com exercícios e sistemas práticos desenvolvidos em Python, aplicando os quatro pilares da Programação Orientada a Objetos: **abstração**, **encapsulamento**, **herança** e **polimorfismo**.

---

## 🧠 Pilares praticados

| Pilar | O que é | Onde foi aplicado |
|---|---|---|
| **Encapsulamento** | Proteger dados internos de uma classe | `encapsulamento.py`, `sistemaBancario.py` |
| **Herança** | Reaproveitar comportamentos entre classes | `formaGeometrica.py`, `sistemaVeiculos.py` |
| **Polimorfismo** | Mesmo método, comportamentos diferentes | `polimorfismo.py`, `zoológico.py` |
| **Abstração** | Modelar entidades do mundo real em classes | Todos os sistemas |

---

## 📁 Projetos e exercícios

### 🏦 Sistema Bancário — `sistemaBancario.py`
Simulação de contas bancárias com operações de depósito, saque e saldo. Aplica encapsulamento para proteger os dados da conta.

### 👨‍💼 Sistema de Funcionários — `sistemaFuncionario.py`
Modelagem de funcionários com diferentes cargos e salários. Usa herança para especializar tipos de funcionários.

### 💳 Sistema de Pagamento — `sistemaPagamento.py`
Diferentes formas de pagamento (cartão, boleto, etc.) com polimorfismo para processar cada tipo.

### 🚗 Sistema de Veículos — `sistemaVeiculos.py`
Hierarquia de veículos com herança — veículo base sendo especializado em tipos específicos.

### 🚌 Sistema de Transporte — `sistemaTransport.py`
Modelagem de meios de transporte aplicando herança e polimorfismo.

### 📦 Produto — `produto.py`
Classe de produto com encapsulamento de preço e quantidade.

### 🔺 Formas Geométricas — `formaGeometrica.py`
Herança entre formas (círculo, retângulo, triângulo) com cálculo de área e perímetro via polimorfismo.

### 🦁 Zoológico — `zoológico.py`
Animais com comportamentos polimórficos — cada animal emite sons e age de forma diferente mesmo compartilhando a mesma interface.

### 🔒 Encapsulamento — `encapsulamento.py`
Exercício direto sobre atributos públicos, privados e protegidos com getters e setters.

### 🔁 Polimorfismo — `polimorfismo.py`
Exercício prático demonstrando sobrescrita de métodos e interfaces comuns entre classes diferentes.
